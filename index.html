    Help

    Total Time Calculation Logic :-

    • Gross (First IN Last OUT): Over all in time for the day
    • Paired Last IN will be considered in the continues INs. first OUT will be considered in the continues OUTs.
    • Flexed This is inverse to Paired. First IN will be considered in the continues INs. Last OUT will be considered in the continues OUTs.

    Settings Menu :-

    • Pair Either In or OUT will be allowed.
    • Flex Both IN and OUT options can be used anytime.

    Buttons :-

    • Settings Shows settings menu.
    • List To load entries of particular date or remove a date from the date list which you have already created.
      • Back To close the date list view and also to cancel the edit date list.
      • Edit To remove any particular date or all dates from your checked-in history.
    • Home To reload today's entries in main view.
    • Edit Can be used to any particular entry or all entries for the selected date.
